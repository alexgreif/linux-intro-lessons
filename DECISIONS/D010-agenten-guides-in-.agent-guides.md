# D010: Detaillierte Agenten-Guides liegen in `.agent-guides/`

Datum: 2026-06-04

## Entscheidung

Detaillierte didaktische Anleitungen für wiederkehrende Dokumenttypen liegen in `.agent-guides/`.

## Begründung

`AGENTS.md` soll schnell scanbar bleiben.
Gleichzeitig sollen die durch Iteration entstandenen Präferenzen detailliert dokumentiert werden, damit neue Übungen mit weniger Überarbeitungsschritten entstehen.

## Konsequenzen

`AGENTS.md` verweist auf die Guides.
Neue detaillierte Regeln zu Kurzreferenzen oder Aufgabenblättern gehören in die passenden Dateien unter `.agent-guides/`.
