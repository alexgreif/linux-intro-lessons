# Übung: Ubuntu-Server-VM einrichten

In dieser Übung richtest du in Hyper-V eine Ubuntu-Server-VM ein.
Du lädst das Ubuntu-Server-ISO herunter, erstellst die VM, installierst Ubuntu Server und sicherst den Grundzustand mit einem Checkpoint.

Die VM ist die Arbeitsumgebung für die späteren Linux-Server-Übungen.

## Lernziele

- eine Ubuntu-Server-VM in Hyper-V aus einem ISO-Abbild erstellen
- grundlegende VM-Ressourcen passend auswählen und dokumentieren
- virtuelle Switches und Netzwerkanschluss der VM auf Überblicksebene einordnen
- Checkpoints sinnvoll einsetzen und von Backups unterscheiden
- eine einfache technische Einrichtung nachvollziehbar dokumentieren
