# Aufgabe 1: Ubuntu-Server-VM einrichten

In dieser Aufgabe richtest du eine Ubuntu-Server-VM in Hyper-V ein.
Die VM ist deine Arbeitsumgebung für die späteren Linux-Server-Übungen.

Nutze dafür die Setup-Hilfen zu [Hyper-V](Hyper-V.md) und [Ubuntu Server](Ubuntu_Server.md).
Fülle während der Einrichtung die [Dokumentationsvorlage](Dokumentationsvorlage.md) aus.

Am Ende sollst du:

- eine startfähige Ubuntu-Server-VM haben
- dich mit deinem Benutzer anmelden können
- einen Checkpoint der Grundinstallation erstellt haben
- die wichtigsten VM-Daten dokumentiert haben

## Arbeitsauftrag

**1.** Lade das Ubuntu-Server-ISO herunter.

**2.** Erstelle in Hyper-V eine VM mit den empfohlenen Einstellungen aus `Hyper-V.md`.

**3.** Installiere Ubuntu Server in der VM.

**4.** Melde dich nach der Installation mit deinem Benutzer an.

**5.** Führe den ersten Netzwerk- und Paketcheck aus und installiere `tree` und `unzip`.

**6.** Erstelle einen Checkpoint der Grundinstallation.

**7.** Vervollständige die Dokumentationsvorlage und notiere gegebenenfalls offene Punkte.
