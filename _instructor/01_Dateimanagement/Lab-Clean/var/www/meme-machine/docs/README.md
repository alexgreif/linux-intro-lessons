# Meme Machine

This directory contains the files for a small static website served by nginx.

The website has three public pages:

- `index.html`
- `about.html`
- `contact.html`

The main page loads random content from the asset folders:

- `assets/jokes/` contains text files with jokes.
- `assets/memes/` contains image files with memes.
- `assets/css/` contains stylesheets.
- `assets/js/` contains JavaScript files.

The files `robots.txt`, `sitemap.xml`, and `.well-known/security.txt` are common website metadata files.

The file `docs/credits.txt` lists sources and licenses for jokes, memes, and other media copied from external websites.

This website is intentionally simple because it is used for Linux and file-management practice.
