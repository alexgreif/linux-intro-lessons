# Übung: SSH praktisch nutzen

In dieser Übung nutzt du SSH, um dich von deinem Host-System aus mit deiner Ubuntu-Server-VM zu verbinden.

SSH ist eines der wichtigsten Werkzeuge für die Arbeit mit Linux-Servern.
Statt direkt an der Server-Konsole zu arbeiten, kannst du damit eine Remote-Shell-Sitzung über das Netzwerk öffnen.

## Lernziele

- die IP-Adresse einer Linux-Server-VM für eine SSH-Verbindung herausfinden
- eine SSH-Verbindung mit Benutzername, IP-Adresse und Passwort herstellen
- die Host-Key-Meldung beim ersten Verbindungsaufbau auf Überblicksebene einordnen
- lokale und entfernte Shell-Sitzungen unterscheiden
- mehrere SSH-Sitzungen erkennen und gezielt beenden
